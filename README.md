# ipl_analysis
Analyzed IPL match data from (2008-2019) using python, pandas, matplotlib and data from kaggle(IPL complete dataset)

## What i explored
-> WHich team won the most matches
-> Does winning toss help win the match
-> How many matches were played per season

## Tools i used
-> Python
-> Pandas
-> Matplotlib
-> Dataset: IPL Complete Dataset (kaggle)
